# git procdure

## Check your current setup 
git remote -v



1. Initialize Git

git init

2. Add remote origin (if not already added)

git remote add origin https://github.com/uzmasecrets-hub/Fastapi-course.git

2. Add files

git add .

3. Commit

git commit -m "first commit"

4. Push

git push -u origin main
