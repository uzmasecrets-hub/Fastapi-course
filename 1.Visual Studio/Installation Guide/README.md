# Visual Studio Code Installation Guide

This guide explains how to install **Visual Studio Code (VS Code)** step by step.  
VS Code is a free, lightweight, and powerful code editor used for Python, FastAPI, Web Development, and more.

---

## What is Visual Studio Code?

**Visual Studio Code (VS Code)** is a modern source-code editor developed by Microsoft.  
It supports multiple programming languages and provides features like:
- Syntax highlighting
- Extensions
- Integrated terminal
- Git support

---

## System Requirements

- Windows 10 or later  
- macOS  
- Linux  
- Internet connection  

---
